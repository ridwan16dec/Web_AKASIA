# PT. Aluminium Kreasi Indonesia Landing Page

Landing page React + Vite untuk PT. Aluminium Kreasi Indonesia, fokus pada produk Bedhead Panel aluminium untuk kebutuhan alat kesehatan.

## Menjalankan Project

```bash
npm install
npm run dev
```

## Build Production

```bash
npm run build
```

## Stack

- React
- TypeScript
- Vite
- Lucide React Icons
